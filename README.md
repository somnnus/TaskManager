1. Проект реализован на **.Net 8**
2. Для запуска необходимо:
    * подгрузить необходимые nuget пакеты
    * собрать приложение
    * положить файл taskmanager.db из "BD File"\taskmanager.db в TaskManager\bin\Debug\net8.0-windows
